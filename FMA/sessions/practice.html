Qfarë është html?

Html ose Hypertext Markup Language,
është një gjuhë markup për web-in që përcakton strukturën e faqeve web.

Vetëm kjo strukturë nuk mjafton për ta bërë një faqe web të duket e mirë dhe interaktive. Kështu që do të përdorni teknologji të asistuara si CSS dhe JAVASCRIPT për ta bërë HTML-në tuaj të bukur dhe për të shtuar interaktivitetin, respektivisht.
